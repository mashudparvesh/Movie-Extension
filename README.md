# 🎬 IMDB Play Now – Chrome Extension

একটি ক্রোম এক্সটেনশন যা IMDB সিনেমা পেজে গিয়ে **Watch Now** বাটনে ক্লিক করলে **৩টি অ্যাড ট্যাব** ও **১টি মূল সিনেমা লিংক** একসাথে খোলে।

## 📥 ডাউনলোড ও ইন্সটল করার নিয়ম

> ⚠️ **শুধুমাত্র ডেস্কটপ ক্রোম ব্রাউজারের জন্য**

1. নিচের লিংক থেকে `.crx` ফাইলটি ডাউনলোড করুন:  
   👉 [**Movie-Ex.crx (সরাসরি ডাউনলোড)**](https://github.com/mashudparvesh/Movie-Extension/raw/refs/heads/main/Movie-Ex.crx)

2. ক্রোম ব্রাউজারের ঠিকানায় `chrome://extensions` লিখে এন্টার দিন।

3. উপরের ডানকোণায় **Developer mode** চালু করুন (অবশ্যই চালু করতে হবে)।

4. ডাউনলোড করা `Movie-Ex.crx` ফাইলটিকে `chrome://extensions` পেজের ভিতরে টেনে এনে ফেলে দিন (Drag & Drop)।

5. "Add extension" বাটনে ক্লিক করুন।

6. এক্সটেনশনটি টুলবারে চলে আসবে ✅

## 🎯 কীভাবে ব্যবহার করবেন?

1. যেকোনো ট্যাবে এক্সটেনশন আইকনে ক্লিক করুন → **Visit IMDB** বাটন দেখাবে।
2. সেখান থেকে যেকোনো সিনেমার পেজে যান (URL হবে `imdb.com/title/tt....`)।
3. আবার এক্সটেনশন আইকনে ক্লিক করুন → এবার দেখাবে **Watch Now** বাটন।
4. **Watch Now** বাটনে ক্লিক করলেই :
   - আসল সিনেমা Watch করতে পারবেন | 
> নোট: প্রথমবার ব্যবহারের সময় ক্রোম অনুমতি চাইতে পারে – "Allow" করে দিন।


## ⚠️ সতর্কতা

- এক্সটেনশনটি ক্রোম ওয়েব স্টোরে নেই। তাই Developer mode চালু করে ইন্সটল করতে হবে।
- এটি সম্পূর্ণ ফ্রি ও ওপেন সোর্স (শুধু লার্নিং ও ব্যক্তিগত প্রজেক্টের জন্য)।
